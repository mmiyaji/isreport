ダウンロード/閲覧

本サイトでは ISレポートをHTML・Epub（電子書籍）・PDF形式でダウンロード/閲覧することが出来ます．

レポート一覧に現れる下記のアイコンをクリックすることでダウンロード出来ます．

  • HTML形式でダウンロード/閲覧する

    通常，ブラウザでの閲覧にはこちらを選択して下さい． レポートタイトルをクリックすることでも閲覧可能です．

  • Epub（電子書籍）形式でダウンロード/閲覧する

    iPadなどが採用している電子書籍フォーマットである Epub形式でダウンロード出来ます． 通常，閲覧には専用のビューアが必要となります．

  • PDF形式でダウンロード/閲覧する

    一部のISレポートには対応していません．

レポート検索

ISレポート検索方法は下記の通りです．

  • サイト内検索

    ・検索バー

    本サイト全体をキーワード検索する場合は こちらをご利用下さい． レポートの中身を検索する場合はこちらが便利ですが，最新のレポートについては検索結果に反映されない場合があります．ご了承下さい． 左のサイドバーに常に表示されています．

    ・検索結果例

  • 研究グループ別検索

    研究グループ別にレポートを検索します．

  • 著者別検索

    著者別にレポートを検索します．このページでは著者が役職別に分類されています．

  • 年度別検索

    年度別にレポートを検索します．このページでは著者が所属年度別に分類されています．

  • タグ別検索

    レポートに付けられたタグ別にレポートを検索します．

ユーザ登録

ISグループメンバーのみユーザ登録をすることが出来ます．ユーザ登録をすることで通常の機能に加えていくつかの機能を利用することが出来ます．

  • ・ユーザ登録/ログイン

    新規ユーザ登録時にはISグループメンバにのみ配布しているSecret Keyを入力する必要があります．

  • ・ユーザ設定

    新規ユーザ登録後，以下の画面にリダイレクトされるのでNameセレクトボックスから自分に該当する名前を選ぶ． イメージや自己紹介の欄は任意で入力する．

  • ・未登録ユーザ

    ・登録ユーザ

    登録ユーザには上記の様に，アップロード･編集・Book機能などが付加される．

アップロード ※登録ユーザのみ

  • 登録したユーザのみ利用可能です． アップロード時に公開/下書き保存を選択できます．公開ボタンを押した時点では最新ISレポート一覧には追加されず，管理人の承認待ち状態となります． 自分の作成したレポートはダッシュボードから参照することが出来ます．

お気に入り/My Book機能 ※登録ユーザのみ

  • 登録したユーザのみ利用可能です． 本サイトでは 複数のISレポートをまとめて一冊のEpub形式の本としてダウンロードする機能があります． その機能を利用する為にはレポートをお気に入りもしくはMy Bookに登録する必要があります． 本の出力はサイドバーのダウンロードから行えます．

その他

  • お気に入りに追加 ※登録ユーザのみ

    登録したユーザのみ利用可能です． 選択したレポートをお気に入りレポートに追加します．

  • My Bookに追加 ※登録ユーザのみ

    登録したユーザのみ利用可能です． 選択したレポートをMy Bookに追加します．

  • 編集 ※登録ユーザのみ

    登録したユーザのみ利用可能です． 全てのISレポートに対して登録ユーザは編集権限を持ちます．

IS記法

IS記法はISレポート作成を補助する文章記述方式です．詳細を以下に記載します．

  • 章を追加

    行頭に「*」で表す 入力) * section 出力) 1 section

  • 節を追加

    行頭に「**」で表す 入力) ** sub section 出力) 1.1 sub section

  • 項を追加

    行頭に「***」で表す 入力) *** subsub section 出力) 1.1.1 subsub section

  • 番号付きリストを追加

    行頭に「+」で表す 入力) + list 出力) 1. list

  • 番号なしリストを追加

    行頭に「-」で表す 入力) - list 出力) ・ list

  • 引用符を追加

    文章を「>? ?<」で囲む

  • テーブルを追加

    「||」もしくは「|*|」で表す 入力) | *abc | def | ghi | 出力)
    abc def ghi

  • テーブルキャプション/引用記号を追加

    文頭に「$|」でテーブルキャプション、文中で「 [ [ ] ] 」で引用記号を表す 入力) $| caption 出力) Table.1 caption 入力) ~[[1]]を参照 出力) ~Table.1を参照

  • イメージを追加

    イメージファイルをアップロードして、貼り付けよう 現在はpngファイルのみ対応 貼り付けボタンを押すと、イメージタグを追加する 文頭に「%% image1.png/300」でイメージタグを挿入する /で区切る値には順に イメージファイル名、横幅を入れる

  • イメージキャプション/引用記号を追加

    文頭に「$%」でテーブルキャプション、文中で「 { { } } 」で引用記号を表す 入力) $% caption 出力) Fig.1 caption 入力) ~ { { 1 } }を参照 出力) ~Fig.1を参照

  • 参照を追加

    「(())」で表す 入力) citer((citee)) 出力) citer*1

    References

    *1: citee
